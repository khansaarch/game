BLOCK SHOOTER - GAME STORYBOARD

1

Game Start - Ready to Play

Score: 0
Lives: 3

Player mulai game dengan karakter ship di bagian bawah layar. UI menampilkan score 0 dan lives 3. Background menampilkan bintang-bintang yang berkedip.

MECHANICS:
• Arrow keys untuk movement
• SPACE untuk shoot
• 3 lives system
2

Target Spawning

Score: 0
Lives: 3

Target blocks mulai spawn dari atas layar dengan warna-warna berbeda. Setiap target memiliki rotasi dan kecepatan jatuh yang berbeda-beda.

MECHANICS:
• Max 3 targets on screen
• 2 second spawn interval
• Random colors & rotation
3

Player Shooting

Score: 0
Lives: 3

Player menembakkan peluru dengan menekan SPACE. Peluru bergerak ke atas dengan trail effect berwarna cyan yang menarik.

MECHANICS:
• SPACE key to shoot
• Bullets have trail effect
• Upward trajectory
4

Target Hit - Score!

Score: 10
Lives: 3

Ketika peluru mengenai target, terjadi ledakan partikel yang spektakuler. Score bertambah +10 poin dan target menghilang.

MECHANICS:
• +10 points per hit
• Particle explosion effect
• Target destroyed
5

Missed Shot - Life Lost

Score: 10
Lives: 2

Jika peluru keluar dari layar tanpa mengenai target, player kehilangan 1 nyawa. Lives counter berkurang dan berubah warna menjadi merah.

MECHANICS:
• -1 life per miss
• UI shows life count
• Warning color change
6

Game Over Screen

GAME OVER
Final Score: 50
RESTART
Score: 50
Lives: 0

Setelah 3 miss, game berakhir. Muncul popup Game Over dengan final score dan tombol restart untuk memulai game baru.

MECHANICS:
• Game ends at 0 lives
• Shows final score
• Restart button available
